Printing Images on Terminal

The following project prints PPM images on the terminal. The project is built using Assembly and C. The input is an image in PPM form and the output is a black and white Image on the terminal which is displayed using ASCII characters. Several modes can be applied to the image, namely contrast, inversion or both.

The knight's tour

Designed and implemented a recursive brute-force algorithm in C++ to solve the Knight's Tour problem on an n x n chessboard, with 'n' provided by user input. Developed a program that guides a knight to visit every square on the chessboard exactly once, and outputs a complete solution on the screen.
